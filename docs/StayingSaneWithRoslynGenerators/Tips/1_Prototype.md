# Tips for creating a prototype

- Before starting:
  - Determine how you’ll extract metadata from user code
  - Decide whether you can be attribute based – vastly better performance
  - Consider your target C# versions, frameworks and libraries
- All code that will be output should appear at least once
- It’s essential that you put this under source control
  - It’s the record of your intent
- Later, create a parallel project which uses generated code
